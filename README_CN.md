# 简单聊天应用

一个基于Flask、Socket.IO和Redis构建的实时聊天应用。用户可以加入聊天房间并实时交换消息。

## 特性
- 实时消息传递
- 消息历史
- 简单清晰的界面
- Docker 部署

## 设置
1. 确保您已安装Docker和Docker Compose
2. 克隆仓库
3. 运行 `docker-compose up --build`
4. 访问应用程序 `http://localhost:5000`
